# parser

Модуль будет отвечать за преобразование SQL-подобного текста в типизированные команды для `engine`.

Планируемые части:

- `Tokenizer` - разбиение входной строки на токены;
- `AST` - структуры команд и выражений;
- `Parser` - синтаксический разбор команд задания 0.

Модуль не должен работать с файловой системой, таблицами или выводом в терминал.
