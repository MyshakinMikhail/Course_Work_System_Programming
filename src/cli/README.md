# cli

Модуль будет отвечать за запуск приложения в двух режимах:

- интерактивный режим без аргументов;
- пакетный режим с одним аргументом - путем к SQL-скрипту.

CLI должен читать команды до символа `;`, передавать их в `parser` и `engine`, печатать результат выполнения и продолжать работу после ошибок.
