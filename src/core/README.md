# core

Этот модуль содержит базовые модели данных и реализацию таблицы, которая работает поверх `storage`.

## Что здесь описано

- `FieldType` - тип значения поля (`INT` или `STRING`).
- `Field` - одно значение записи.
- `Record` - строка таблицы, то есть набор `Field`.
- `Column` - описание одного столбца схемы: имя, тип и признак индексирования.
- `TableSchema` - схема таблицы.
- `Table` - конкретная таблица, которая хранит схему, пишет записи в файл и читает их через `FileManager`.

## `Table`

`Table` теперь использует файловое хранилище:

- `initialize()` - открывает файл и записывает схему;
- `insert()` - добавляет запись в `DATA` и сохраняет `offset`;
- `scan()` - последовательно читает все сохраненные записи;
- `findByIndex()` - читает запись по номеру вставки.

## Сборка модуля

`src/core` собирается как библиотека `core` и зависит от `storage`, чтобы использовать `FileManager`.
