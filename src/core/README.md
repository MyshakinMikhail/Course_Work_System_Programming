# core

Этот модуль содержит базовые модели данных, на которых строится вся работа с таблицами.

## Что здесь описано

- `FieldType` - тип значения поля (`INT` или `STRING`).
- `Field` - одно значение записи. Хранит либо целое число, либо строку и умеет удобно создаваться через конструкторы и фабрики `Int` / `String`.
- `Record` - строка таблицы, то есть набор `Field`.
- `Column` - описание одного столбца схемы: имя, тип и признак индексации.
- `TableSchema` - схема таблицы, которая хранит набор столбцов и помогает искать столбец по имени.

Эти структуры используются как основа для хранения данных, работы со схемой таблицы и дальнейшего построения API для вставки и поиска записей.

## Сборка модуля

`src/core` подключается как header-only CMake target. Отдельная цель `core` создаётся в общем `CMakeLists.txt`, а заголовки доступны через include path `src/core`.
