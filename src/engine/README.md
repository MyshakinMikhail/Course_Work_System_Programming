# engine

Модуль будет отвечать за семантическую проверку и выполнение команд, полученных от `parser`.

Планируемые части:

- единый результат выполнения команды;
- каталог баз данных и таблиц;
- диспетчеризация DDL/DML-команд;
- вычисление условий `WHERE`;
- форматирование результата `SELECT` в JSON.

Модуль является границей между SQL-слоем и `core`/`storage`.
