# Course_Work_System_Programming — Системное программирование (Вариант 3: B*-дерево)

Это репозиторий с реализацией упрощённой мини-СУБД, написанной на C++ в рамках курсовой работы по предмету *Системное программирование*.

Проект представляет собой небольшую базу данных с поддержкой:

- файлового хранения таблиц;
- парсинга SQL-подобного языка;
- индексации с помощью **B*-дерева**;
- интерактивного и пакетного интерфейса.

## 🎯 Задача

Реализовать систему управления данными с поддержкой:

- создания таблиц;
- вставки записей;
- выборки по ключу с эффективным поиском через **B*-Tree** индекс.

## 📌 Основные контракты и структуры

### 1. **Core — модели данных**

**Record и Field**

```cpp
enum class FieldType {
    INT,
    STRING
};

struct Field {
    FieldType type;
    std::string stringValue;
    int intValue;
};

struct Record {
    std::vector<Field> fields;
};
```

Это основные структуры, которые описывают запись таблицы и поля в ней.

### 2. **Storage — файловое хранение**

Таблицы хранятся в бинарных файлах с двумя частями:

- схема таблицы;
- данные.

Каждая запись записывается последовательно, а позиция записи (`offset`) используется в индексе.

### 3. **Index — контракт индексации**

Абстрактный интерфейс для всех индексов:

```cpp
class Index {
public:
    virtual void insert(int key, int offset) = 0;
    virtual std::optional<int> find(int key) = 0;
};
```

Индекс связывает ключ, например `id`, с позицией записи в файле.

### 4. **B-Tree — реализация индекса**

В проекте реализован вариант **B*-дерева** — сбалансированного дерева с высокой загрузкой узлов:

```cpp
class BStarTree : public Index {
    struct Node {
        std::vector<int> keys;
        std::vector<int> offsets;
        std::vector<Node*> children;
        bool isLeaf;
    };

    Node* root;
};
```

**B*-Tree** обеспечивает:

- быстрый логарифмический поиск (`O(log N)`);
- минимальную загрузку узлов (`≥ 2/3`);
- перераспределение между соседями перед разделением.

### 5. **Table — API таблицы**

Класс `Table` объединяет данные и индекс:

```cpp
class Table {
public:
    void insert(const Record& record);
    std::vector<Record> scan();
    std::optional<Record> findByIndex(int key);
};
```

При вставке:

- запись пишется в файл;
- получается `offset`;
- обновляется индекс.

При `SELECT`:

- если индекс есть, используется поиск по индексу;
- иначе выполняется полный `scan`.

### 6. **Parser — синтаксический разбор SQL-подобных команд**

Поддерживаются базовые команды:

- `CREATE DATABASE`;
- `CREATE TABLE`;
- `INSERT INTO`;
- `SELECT ... WHERE id = ...`.

Поддерживается разбор минимального синтаксиса для собственной реализации.

### 7. **Engine — исполнение команд**

Движок получает результат парсера и вызывает соответствующие методы API:

| SQL-команда | Что делает |
|---|---|
| `INSERT` | Записывает запись и обновляет индекс |
| `SELECT` | Находит запись через индекс или полный `scan` |

### 8. **CLI — интерфейс пользователя**

Поддерживает два режима:

**Интерактивный**

```bash
./program
> INSERT INTO users VALUES ...
```

**Batch**

```bash
./program script.sql
```
